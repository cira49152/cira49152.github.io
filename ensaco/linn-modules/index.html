LINN Modules

Industrial automation system elements - The Power of modularity

As part of our strategy to provide all necessary building elements for Industrial automation systems, pallete of various LINN Modules are at Customer disposal, as stand alone or part of larger Solution.

Communication protocol - Modbus

Protocol used to establish client-server communication between intelligent devices. It is widely used network protocol in the industrial manufacturing environment. Modbus is used in multiple client-server applications to monitor and program devices; to communicate between intelligent devices and sensors and instruments; to monitor field devices using PCs and HMIs, protocol for RTU applications where wireless communication is required. Both TCP/IP and RTU are supported.

Communication protocol - OPC UA

The OPC Unified Architecture protocol (TCP/IP based)

Communication protocol - ISO

The ISO family of protocols isn't widely used, as the Internet Protocol Family is much more common today. Yet, their core features can be deployed in industrial applications.

Communication protocol - MQTT - Comming Soon

Message Queuing Telemetry Transport is an open OASIS and ISO standard (ISO/IEC 20922) lightweight, publish-subscribe network protocol that transports messages between devices.

Communication protocol - IEC 61850-8-1 - Comming Soon

Standard that describes models and communication protocols used in Smart Grid

Communication protocol - DNP3 - Comming Soon

Distributed Network protocol. It is used in communication between substation computers, RTUs, IEDs (Intelligent Electronic Devices) and master stations (except inter-master station communications) for the electric utility industry.

Historian module - Comming Soon

RX platform with historian functionalities incorporated. It supports data sources natively supported by RX platform (incorporated communication protocols), various database formats (SQL and NoSQL), provides data visualization and reporting, different storage solutions for securing data records.

Security modules - Comming Soon

Security module is built to establish following RX-platform capabilities (NIST SP 1800-23A: Energy Sector Asset Management):

  • OT/ICS asset inventory (including devices using serial connections)
  • high-speed communication mechanisms for remote asset management
  • reliable/secure/encrypted communications
  • continuous asset monitoring
  • log analysis and correlation
  • cybersecurity event/attack detection
  • patch-level information
  • vulnerability awareness

Softlogic module - Comming Soon

Module that enables representing logical equations and simple actions. Support for IEC 61131-3 PLC Languages such as, Sequential Function Chart (SFC), Quick Ladder Diagram (LD), Function Block Diagram (FBD).

Simulation module - Comming Soon

Module specialized to enable versatile tags simulation and its dynamics for the purpose of pretesting and validation of engineering.
